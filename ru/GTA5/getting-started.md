# Начало работы

Используйте эту страницу, чтобы подготовить Banka Majestic Spoofer перед входом в GTA5 Majestic RP.

## Перед началом

- Убедитесь, что используете GTA5 Majestic RP.
- Скачайте архив приложения.
- Подготовьте ключ, полученный после покупки.

## Шаги

1. Распакуйте архив в отдельную папку.
2. Запустите `.exe` файл.
3. Дождитесь полной загрузки приложения.
4. Введите ключ, полученный после покупки.
5. Создайте первый keyset.
6. Выберите keyset, который хотите использовать.
7. Нажмите **Start**.
8. Войдите в игру.

## Результат

Выбранный HWID-профиль применяется к текущей сессии GTA5.

## Важно

- Banka Majestic Spoofer предназначен только для GTA5 Majestic RP.
- HWID spoofing работает только в рамках текущей игровой сессии.
- Чтобы изменить активный keyset, перезапустите игру.
