# Обзор

Banka Majestic Spoofer - инструмент для временной подмены HWID в текущей сессии GTA5. Он предназначен только для серверов Majestic RP.

Изменения применяются только к активной игровой сессии и не меняют системные настройки Windows.

## Основные возможности

- Работает только на серверах GTA5 Majestic RP.
- Временно подменяет HWID только для текущей игровой сессии.
- Не меняет системные настройки Windows.
- Поддерживает keyset-ы - отдельные сохраненные HWID-профили.
- Позволяет создавать, удалять, применять и переключать сохраненные keyset-ы.
- Требует повторного входа в игру после применения другого keyset-а.

## Что такое keyset

Keyset - это сохраненный HWID-профиль. Можно создать несколько keyset-ов и выбрать нужный профиль перед запуском игры.
