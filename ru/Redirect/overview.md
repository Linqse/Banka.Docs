# Обзор

Redirect помогает обходить блокировку ввода с клавиатуры и мыши при удаленном подключении к компьютеру через Parsec.

## Когда использовать

Используйте Redirect, если в целевом окружении заблокирован удаленный ввод, а подключение выполняется через Parsec.

Сейчас Redirect поддерживает только Parsec. Другие инструменты удаленного доступа не поддерживаются.

## Совместимость

Redirect протестирован с основными античит-системами Lineage 2 и корректно работает при соблюдении условий запуска.

## Лицензия

Ключ, выданный после покупки, дает доступ одному пользователю и не ограничивает использование одним ПК.
