# Overview

Redirect helps work around blocked keyboard and mouse input when connecting to a computer remotely through Parsec.

## When To Use

Use Redirect when remote-control input is blocked in the target environment and the connection is made through Parsec.

At the moment, Redirect supports Parsec only. Other remote access tools are not supported.

## Compatibility

Redirect has been tested with the main Lineage 2 anti-cheat systems and works correctly when the launch requirements are met.

## License

The license key issued after purchase grants access to one user and is not limited to a single PC.
