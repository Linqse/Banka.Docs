# Overview

Banka Majestic Spoofer is a tool for temporary HWID spoofing in the current GTA5 session. It is designed exclusively for Majestic RP servers.

The spoofing changes apply only to the active game session and do not modify Windows system settings.

## Key Features

- Works only on GTA5 Majestic RP servers.
- Temporarily spoofs HWID for the current game session.
- Does not change Windows system settings.
- Supports keysets, which are separate saved HWID profiles.
- Allows creating, deleting, applying, and switching between saved keysets.
- Requires re-entering the game when a different keyset is applied.

## What Are Keysets

A keyset is a saved HWID profile. You can create multiple keysets and select the required profile before launching the game.
