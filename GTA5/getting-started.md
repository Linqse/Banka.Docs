# Getting Started

Use this page to prepare Banka Majestic Spoofer before entering GTA5 Majestic RP.

## Before You Start

- Make sure you are using GTA5 Majestic RP.
- Download the application archive.
- Prepare the key received after purchase.

## Steps

1. Extract the archive to a separate folder.
2. Run the `.exe` file.
3. Wait until the application fully loads.
4. Enter the key received after purchase.
5. Create your first keyset.
6. Select the keyset you want to use.
7. Click **Start**.
8. Enter the game.

## Result

The selected HWID profile is applied for the current GTA5 session.

## Important

- Banka Majestic Spoofer is intended exclusively for GTA5 Majestic RP.
- HWID spoofing works only within the current game session.
- Restart the game to change the active keyset.
